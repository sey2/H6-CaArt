## 👬 팀 이름
> 유저는 이 서비스를 통해서 자신의 라이프 스타일에 딱 맞는 나만의 자동차를 설계할 수 있게 됩니다.  
> 사용자의 생활 일부를 그림을 그리듯 자동차로 그려내고 있다고 생각해서 자동차와 예술을 합하여 카아트!
